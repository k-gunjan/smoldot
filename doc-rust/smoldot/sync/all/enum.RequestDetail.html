Variants

BlocksRequest

Fields

first_block_height: u64

Height of the first block to request.

first_block_hash: Option<[u8; 32]>

Hash of the first block to request. None if not known.

ascending: bool

True if the first_block_hash is the response should contain blocks in an increasing number, starting from first_block_hash with the lowest number. If false, the blocks should be in decreasing number, with first_block_hash as the highest number.

num_blocks: NonZeroU64

Number of blocks the request should return.

Note that this is only an indication, and the source is free to give fewer blocks than requested.

This might be equal to u64::max_value() in case no upper bound is required. The API user is responsible for clamping this value to a reasonable limit.

request_headers: bool

True if headers should be included in the response.

request_bodies: bool

True if bodies should be included in the response.

request_justification: bool

True if the justification should be included in the response, if any.

Requesting blocks from the source is requested.
